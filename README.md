# 🏗️ Калькулятор ЛСТК-профилей

Интерактивный калькулятор раскроя гнутых стальных профилей **любых размеров** для расчёта себестоимости и оптимизации отхода.

**Контекст задачи:** Станок для гибки профиля позволяет изготавливать изделия с **шагом 1 мм** по ширине и высоте. Широкая сетка комбинаций размеров, видов профилей, высот стенок, полок и отгибок **не позволяет составить полный прайс** — вариантов слишком много.

---

## ⚠️ Важно для обывателя: что считает этот калькулятор

Это **не простой подсчёт суммы высоты полок и отгибок**.

Калькулятор выполняет **инженерный расчёт развёртки** с учётом реальной геометрии гибки оцинкованного рулона:

### 🔬 Ядро расчёта

**Входные данные (тип профиля + размеры):**
- Высота стенки (H): 100–350 мм
- Ширина полки A: 40–100 мм
- Ширина полки B: 40–95 мм (для ПЗ)
- Отгибка C: 13–27 мм
- Толщина оцинкованного рулона (t): 1.0–3.0 мм

**Ключевые формулы расчёта:**

```
Радиус внутреннего изгиба (rb):      по толщине из СП 16.13330
Нейтральный радиус (rcp):            rb + t / 2

Кол-во изгибов (bendsCount):
  • ПП = 2   • ПГС = 4   • ПЗ = 4

Длина изогнутых участков (l0):
  l0 = bendsCount × (π × rcp / 2)

Длина прямолинейных участков (l1):
  • ПП:   H − 2×(rb + t) + 2×(A − (rb + t))
  • ПГС:  H − 2×(rb + t) + 2×(A − 2×(rb + t)) + 2×(C − (rb + t))
  • ПЗ:   H − 2×(rb + t) + (A − 2×(rb + t)) + (B − 2×(rb + t)) + 2×(C − (rb + t))

Полная развёртка (razvertka):        l0 + l1

Количество из рулона:
  rollWidth = 1250 мм (t ≤ 2.0)
  rollWidth = 1000 мм (t ≥ 2.5)
  countFromRoll = ⌊rollWidth / razvertka⌋

Отход:                               rollWidth − countFromRoll × razvertka
Цена с отходом:                      распределение стоимости рулона
```

### 📐 Параметры материала по толщине (СП 16.13330)

| t (мм) | rb (мм) | Ширина рулона | Уд. вес (кг/м²) |
|---|---|---|---|
| 1.0 | 3.0 | 1250 | 7.5056 |
| 1.2 | 3.2 | 1250 | 9.2304 |
| 1.5 | 3.5 | 1250 | 11.852 |
| 2.0 | 4.0 | 1250 | 15.67696 |
| 2.5 | 4.5 | **1000** | 19.62 |
| 3.0 | 5.0 | **1000** | 23.5 |

### 🎯 Типы профилей

- **ПП** — профиль П-образный (2 изгиба)
- **ПГС** — профиль П-образный с отгибками (4 изгиба)
- **ПЗ** — профиль Z-образный (4 изгиба, асимметричный)

## 🔥 Уникальная фича: Heatmap отходов

Для данной толщины и типа профиля строится визуальная карта отхода при разных сочетаниях высоты и ширины полки:

![Пример heatmap](assets/screenshots/waste-heatmap.png)

**Зачем проектировщику:**
- Один взгляд → понятно, какие комбинации дают **минимальный отход**
- Шаг сетки: 10 мм по высоте, 5 мм по ширине
- Цветовая шкала: 🟢 ≤5% → 🟡 ≤10% → 🟠 ≤20% → 🔴 ≤35% → 🟣 >35%

## 🎓 Для работодателя / рекрутера

Этот проект показывает:

1. **Domain knowledge:** 18 лет в строительных материалах = глубокое понимание процесса гибки профилей и нормативных требований (СП 16.13330).
2. **Перевод бизнес-логики в код:** менеджерская задача «назови цену за 30 секунд» переведена в точный инженерный расчёт.
3. **Сложные вычисления:** геометрия сгиба, радиусы изгиба, компенсация длин, оптимизация раскроя.
4. **UX/UI для нетехнических пользователей:** менеджер не должен знать формулы — он видит результат за два клика.
5. **Оптимизация:** heatmap помогает проектировщику выбрать размеры с минимальным отходом = экономия 15–30% материала.
